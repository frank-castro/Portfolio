About this site;

SPA built in AngularJS, with Sass (Modular-scale, Susy) and Compass, HTML5, CSS3, SVG graphics, Grunt, Git, Github. Angular-parallax for scrolling effect, and Slick.js for touch / swiping effect. All content is generated dynamically from JSON files.
