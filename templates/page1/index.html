Welcome to Frank’s personal website.

Frank is an experienced designer and Creative Director living in Ottawa, Canada. He has designed many things in the past but nowadays he mainly focuses on web design.

He likes to code his own designs so he builds prototypes using AngularJS or Javascript. Sometimes he thinks of himself as a programmer but he is not... he can not code without Stackoverflow.

In more specific terms, Frank defines the quality of the user experience, the art direction, design the visual elements, and implement / manage the front end development of web applications for different platforms and devices.

He is currently employed at Phirelight as Creative Director responsible for the design and prototyping of RapidPhire; the company’s network security software solution.
